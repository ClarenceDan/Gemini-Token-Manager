API 调用教程

GeminiAPI负载均衡服务让您可以通过统一的入口访问基于 API Key 的服务，自动为您选择可用的 API Key，实现负载均衡与高可用。

基本使用

调用方式与原始API完全相同，只需将请求地址修改为我们的代理地址，并在请求头中使用您的专属API Key。

示例请求
# 向负载均衡服务发送请求 curl -X POST '/v1/chat/completions' \\ -H 'Content-Type: application/json' \\ -H 'Authorization: Bearer ${authToken}' \\ -d '{ "model": "Qwen/Qwen2.5-7B-Instruct", "messages": [{"role": "user", "content": "你好"}], "stream": true }'

支持的端点

我们支持Gemini的所有API端点，您只需将原始API地址替换为我们的代理地址即可：

端点替换示例
# 原始API地址 https://api.siliconflow.cn/v1/... # 替换为 /v1/...

获取访问授权

要使用该服务，您需要申请专属的API Key。请联系管理员获取访问权限：

管理员邮箱：admin@killerbest.com

主要优势

  • 自动负载均衡 - 系统自动选择有效的API Key
  • 提高可用性 - 单个Key故障不影响整体服务
  • 简化管理 - 无需手动切换API Key
  • 统一入口 - 使用一个API Key访问所有服务
